# Run and deploy your app

This contains everything you need to run the app locally.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env.local` or `.env`
3. Set these values:
   `JWT_SECRET`
   `ADMIN_USERNAME`
   `ADMIN_PASSWORD`
4. Run the app:
   `npm run dev`
